Présentation

Ce framework (pour l'instant sans nom faut pas déconner) est un framework CSS / Javascript, qui utilise Less pour la compilation, et met à disposition des fonctions intégrées dans ce langage, il est donc conseillé d'utiliser ce framework CSS afin d'utiliser au mieux ce ce dernier.

NB : Au passage, les bouts de code de l'in-depth sont aussi en Less, comme ça c'est dit.

Les containers

Je pense que vous connaissez Bootstrap. Et bien moi pas très bien, mais je pense qu'il existe cette notion aussi. Les containers sont des trucs qui en contiennent d'autres. Il y a deux types de containers :

Row

Si vous êtes bon en anglais, vous comprendrez qu'une row est une liste horizontale. C'est une 'rangée' pour être plus précis. Sans arguments, une row ressemble à ça :

Ceci
est
une
row
Voir le code

<div class="row">
	<div class="black white-t padding-S">Ceci</div>
	<div class="black-light white-t padding-S">est</div>
	<div class="white-dark black-t padding-S">une</div>
	<div class="white black-t padding-S">row</div>
</div>

Column

A l'inverse de son fier camarade row, column range les éléments de haut en bas (comme une colonne haha). Un petit exemple :

Ceci
est
une
column
Voir le code

<div class="column">
	<div class="black white-t padding-S">Ceci</div>
	<div class="black-light white-t padding-S">est</div>
	<div class="white-dark black-t padding-S">une</div>
	<div class="white black-t padding-S">column</div>
</div>

Le positionnement du contenu

Pour positionner le contenu dans ces containers, il suffit d'y ajouter comme argument le positionnement vertical, puis horizontal. Par exemple, nommer sa row : row-t-l fera en sorte que la row positionne son contenu tn haut (-t) à gauche (-l). Exemples :

column-b-c :

Verticalement
en bas
Horizontalement
centrés

row-c-c :

Verticalement
centrés
Horizontalement
aussi
Voir le code

Bas-centré :


<div class="column-b-c border padding-S">
	<div class="blue white-t padding-S">Verticalement</div>
	<div class="blue-dark white-t padding-S">en bas</div>
	<div class="black-light white-t padding-S">Horizontalement</div>
	<div class="black white-t padding-S">centrés</div>
</div>

Centré-centré :


<div class="row-c-c padding-S border h100 w450">
	<div class="blue white-t padding-S">Verticalement</div>
	<div class="blue-dark white-t padding-S">centrés</div>
	<div class="black-light white-t padding-S">Horizontalement</div>
	<div class="black white-t padding-S">aussi</div>
</div>

Fonctions usuelles

Les fonctions usuelles sont des fonctions qui peuvent s'appliquer à n'importe quel objet.

Padding / Margin

Pour ceux qui ne sont pas dans le développement web, le padding représente la marge intérieure et margin la marge exterieure.

Afin d'appliquer ces attributs à vos éléments, ils vous suffit de donner l'attribut à modifier, suivi de la taille souhaitée. Par défaut, les tailles sont comme suit :

NomValeur
-s5px
-S10px
-m15px
-M20px
-l25px
-L30px

Ce qui donne par exemple padding-S pour définir une 'petite' marge interieure, ou bien margin-L pour définir une grande marge interieure. En savoir plus

Exemples avec leur code :
padding-S
Mon papa a du padding

Code :


<div class="padding-S shadow">
	<div class="blue white-t">Mon papa a du padding</div>
</div>
margin-L
Mais la j'ai du margin (et du padding)

Code :


<div class="shadow">
	<div class="blue white-t margin-L padding-S">Mais la j'ai du margin (et du padding)</div>
</div>

Espacement

Par espacement, j'entend l'espace entre les élements contenus. Il y a plusieurs manières de régler ça :

Space

space utilise les même valeurs par défaut que margin et padding, et s'emploie de la même manière. Elle définit l'espace (de tous les côtés) qu'il doit y avoir entre les éléments. Il est aussi possible de choisir le sens d'espacement.
